Guide to use this project..

To use this todo you should have to add project first. Go in the Add to project section and click plush icon and it help to push the project to the project folder after that you can hide and unhide the project clicking the icon next to the project. After that select the + icon in the bottom of the page to add todo for different project. And that Today Task and Upcoming Task cannot be deleted it is default.
